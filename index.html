Current State of the Application

Overview

FlashyCash is a decentralized application (DApp) built to facilitate various cryptocurrency transactions and services. The backend is developed using the Tornado web framework and interacts with Ethereum nodes via the web3.py library.

Features

Buy

Users can purchase cryptocurrency by connecting their MetaMask wallet. The service adapts to the userâ€™s region to ensure compliance with local regulations.

Sell

Users can sell their cryptocurrency at current market rates, facilitated through their connected MetaMask wallet.

Send

Users can send cryptocurrency to others using private and public addresses. This feature includes prompts for address input.

Swap

Allows users to check and exchange the current value of their cryptocurrency holdings.

Mobile Top-up

Enables users to recharge their mobile phones with cryptocurrency, compatible with various network services.

Gift-Card

Allows users to purchase and recharge gift cards for various companies using cryptocurrency.
